1. 系统简介

Protein Evaluator 是一个专业的蛋白质结构与功能综合评估系统，基于 UniProt、PDB、RCSB 等公共数据库，结合 AI 大模型生成专业的蛋白质评估报告。

核心功能

  • 🔬 UniProt 元数据获取 - 获取蛋白名称、基因、物种、序列、功能描述等
  • 🗼 PDB 结构分析 - 获取实验方法、分辨率、文献引用等详细信息
  • 🔍 BLAST 同源搜索 - 搜索相似蛋白，分析同源性
  • 🧠 AI 智能分析 - 调用 AI 大模型生成综合分析报告
  • 📄 多格式导出 - 支持 PDF、Markdown 格式导出
  • ⚡ 批量评估 - 支持多个蛋白批量评估和蛋白互作网络分析
💡 支持的 AI 模型

支持 OpenAI (GPT-4o)、Anthropic Claude、Gemini 等主流 AI 模型，可通过配置切换

2. 快速开始

只需 3 步即可完成一次蛋白质评估：

第 1 步：配置 AI

点击右上角「AI 配置」按钮，设置您的 API Key 和模型。配置保存在浏览器本地。

第 2 步：输入 UniProt ID

在输入框中输入要评估的 UniProt ID（如 P04637、TP53_HUMAN），点击「开始」

第 3 步：查看结果

等待评估完成后，查看 AI 生成的综合分析报告，可选择导出 PDF 或 Markdown

📝 有效的 UniProt ID 示例

P04637 (TP53), P00533 (EGFR), P00720 (Lysozyme), P68871 (Hemoglobin subunit beta)

4. 单个蛋白评估

对单个蛋白质进行全面的结构与功能评估。

操作步骤

  1. 确保在「单个评估」Tab 下（默认）
  2. 在输入框中输入 UniProt ID
  3. 点击「开始」按钮
  4. 观察进度条，等待评估完成
  5. 查看评估结果
输入框占位符 输入 UniProt ID
当前模型显示 当前模型: deepseek / deepseek-chat

评估流程

进度 步骤 说明
0% 等待开始 准备评估环境
30% 获取 UniProt 数据 蛋白名称、基因、物种、序列、功能
50% 获取 PDB 数据 结构方法、分辨率、文献
70% BLAST 搜索 同源蛋白分析
90% AI 分析 调用 AI 生成报告
100% 完成 结果展示

5. 批量评估

同时评估多个蛋白质，并分析蛋白之间的相互作用网络。

操作步骤

  1. 点击「批量评估」Tab 切换到批量模式
  2. （可选）输入任务名称
  3. 输入多个 UniProt ID，用空格、逗号或换行分隔
  4. 点击「开始」
  5. 查看批量评估进度和结果
📝 ID 分隔示例

以下三种方式均可：
P04637 P00533 P00720
P04637, P00533, P00720
P04637
P00533
P00720

批量评估结果

  • 📊 综合关系分析报告 - AI 分析多个蛋白之间的相互作用
  • 🔗 蛋白互作网络 - 从 String Database 获取的相互作用信息
  • 📋 单独评估结果 - 每个蛋白的详细评估报告

7. 模板管理

点击「模板管理」按钮可以自定义 AI 提示词模板。

模板类型

  • 📄 单个蛋白评估模板 - 控制单个蛋白评估的 AI 分析逻辑
  • 🔗 批量关系分析模板 - 控制批量评估中的蛋白互作分析

模板语法

使用 {outline} 作为数据插入位置的占位符。例如：

请分析以下蛋白质数据：
{outline}

请生成一份详细的分析报告，包含：
1. 蛋白质功能描述
2. 结构特征分析
3. 研究建议

操作

  • 新建模板 - 创建新的提示词模板
  • 编辑 - 修改现有模板内容
  • 设为默认 - 将模板设为默认使用
  • 删除 - 删除不需要的模板

10. 常见问题

Q1: 如何获取 API Key？

请访问对应 AI 服务的官网注册账号并获取 API Key：

  • OpenAI: https://platform.openai.com/
  • DeepSeek: https://platform.deepseek.com/
  • Anthropic: https://www.anthropic.com/

Q2: 评估进度卡住不动怎么办？

检查网络连接，确保可以访问 UniProt、PDB 等外部 API。如果问题持续，可以查看详细日志了解具体错误信息。

Q3: 为什么 PDB 数量显示为 0？

该蛋白质可能没有对应的 PDB 结构，或者 PDB ID 列表未公开。

Q4: 批量评估支持多少个蛋白？

建议一次批量评估不超过 20 个蛋白，过多可能会导致处理时间过长或 API 超限。

Q5: 报告生成失败怎么办？

常见原因：

  • API Key 配额不足
  • 网络连接问题
  • 蛋白质数据异常

请检查 AI 配置设置，或尝试重新评估。

Q6: 模板中的 {outline} 是什么？

{outline} 是系统占位符，会被自动替换为收集到的蛋白质数据（UniProt、PDB、BLAST 等）。
